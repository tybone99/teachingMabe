Box-shadow Syntax:

box-shadow: offset-x offset-y [blur-radius] [spread-radius] [color]

Multiple Shadows

Use a comma to add additional shadows

box-shadow: 5px 10px, -10px 6px #662C76

Inset Shadows

Use the term "inset" to put the shadow on top of the element, as if the element is sinking into the page

box-shadow: 2px 2px 8px 2px #777 inset
box-shadow: 2px 2px 8px 2px #777 inset

All Together Now!

Translucent

By using a "washed-out" color that is similar to your element and a large blur, the shadow makes it look like your element is slightly translucent, allowing light to pass behind it.

box-shadow: 5px 10px 10px #9FC59A

Straight Back

Our eyes see things smaller the farther back they are. To give the impression that an element is hovering over the page, create a shadow that is smaller than the element, using a negative spread. This generally means that you'll also have to shift it down, since spread affects both the x and y directions.

box-shadow: 0 10px 8px -5px #666

Fuzzy Boundaries

By using no offset, we can create a soft border around our element. Make sure the color is the same as the element, and the spread is half of the radius.

By itself, this looks pretty ugly, but you can use different amounts blur to create a "focal lense" effect with multiple elements. More blur means the element is farther from the focal plane, less means it's closer. (You can get the same effect for the text using text-shadow in the same way)

box-shadow: 0 0 8px 4px #38A02C

Using this idea, we can create a highlight using a different color, like this (generally used with :hover)

box-shadow: 0 0 20px 5px #0367EA

Long-cast Shadow

By using very large values for the offset and spread, you can create a shadow as if it is cast 10 feet back from the object. Just make sure you have enough room.

If you animate the color to be darker and lighter, it looks like an inconsistent light source. Match this with an animation on the element's background-color to give it a more washed-out color when the light goes brighter.

box-shadow: 0 0 8px 4px #38A02C
